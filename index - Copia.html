Curso de Infraestrutura de TI Com Windows Server 2012 R2!

Descrição:
Neste curso apresento de forma clara e objetiva, como trabalhar com o Windows Server 2012 R2. Iremos entender as principais características do sistema, realizar a instalação do tipo GUI e Server Core, realizar a alteração entre tipos de instalação de GUI para Server Core e de Server Core para GUI. Promover um controlador de domínio principal, promover um controlador de domínio adicional, adicionar um File Server e configurá-lo, Gerenciamento de discos, trabalhando com Raid-5, Configurar DHCP, Trabalhar com GPO, Adicionar máquinas ao domínio, adicionar servidores ao domínio utilizando o tipo de instalação Server Core, Realizar configurações em servidores Server Core, Instalando recursos e funções remotamente pelo Server Manager, Trabalhar com o powershel, Integrar nossa infraestrutura com o firewall pfsense.

Curso de Infraestrutura de TI com Samba4 + AD + Pfsense + File Server!

Descrição:
Este curso ensina passo a passo como você planejar uma infraestrutura contendo um PfSense liberando o DHCP e internet para um controlador de domínio principal Debian 9.5 com samba 4, um controlador de domínio secundário Windows Server 2008 R2, um controlador de domínio secundário Debian 9.5 com samba 4 e um File Server dedicado utilizando o Debian 9.5 com samba 4. Trabalharemos com GPO, provisionaremos um servidor WEB, utilizaremos um servidor Centos7 no domínio totalmente integrado com o AD para realização de backups na rede utilizando a ferramenta bacula. Ainda configuraremos um servidor para monitoramento da infraestrutura com zabbix e faremos a integração do mesmo para autenticação dos usuários através do AD. Também veremos como colocar máquinas windows 7, 8 e Ubuntu 18.04 no domínio samba 4. Trabalharemos na linha de comando e através do RSAT para gerenciar o AD. Além disso, o curso conta com um módulo específico para os usuários iniciantes em linux, para que qualquer usuário, mesmo sem conhecimento prévio de linux, possa acompanhar o conteúdo.

Descrição Este curso é voltado para pessoas que querem aprender a trabalhar com AD, GPO e File Server utilizando o samba 4.9.4 e Debian 9.8.0. Também mostro as principais funções do samba-tool. Perguntas Frequentes!

Caso ainda tenha alguma dúvida, veja se a mesma pode ser sanada abaixo. Selecionamos as principais dúvidas dos possíveis clientes antes de adquirir um produto ou serviço.
1. Caso eu não goste do curso posso pedir ressarcimento?

Sim. Você tem até 30 dias para pedir ressarcimento do valor integral que foi pago pelo treinamento.

2. Terei direito a certificado ao concluir o curso?

Todos os cursos pagos possuem certificados de conclusão que variam de acordo com a quantidade de aulas. Para poder imprimir o certificado o aluno precisará assistir todas as aulas e realizar todos os exercícios propostos.

3. Tenho direito a certificado em cursos grátis?

Infelizmente a Udemy não emite certificados para cursos gratuitos no Brasil.

4. O cupom de desconto que tenho está vencido. O que posso fazer para conseguir outro?

Neste caso entre em contato conosco pelo email abaixo que disponibilizamos cupons de desconto para você.

suporte@edrsr.com.br.

5. Por quanto tempo terei acesso ao conteúdo dos cursos?

A Udemy garante acesso vitalício ao conteúdo do curso.

6. Como posso assistir as aulas?

Você pode assistir as aulas através do computador ou pelo aplicativo no Smartphone. No Smartphone você também pode assistir offline, ou seja, quando estiver no wifi você baixa o conteúdo e assiste em outro lugar que não tenha internet sem gastar seus dados móveis. Neste caso você só precisará da internet para logar na conta, depois pode desabilitá-la e assistir as aulas a vontade.
